4
1. add - Adds support for an external library.
   Ex: ng add @angular/material
2. analytics - disable usage of annonymous xcollection data
   Ex: ng analytics off
3. build (b) - Compiles the app to dist/.
   Ex: ng build --prod
4. cache - Configures/retrieves disk cache stats.
   Ex: ng cache status
5. completion - Enables CLI autocompletion.
   Ex: ng completion >> ~/.bashrc
6. config - Gets/sets values in angular.json.
   Ex: ng config cli.defaultCollection @angular-eslint/schematics
7. deploy - Invokes deploy builder.
   Ex: ng deploy
8. e2e (e) - Builds, serves & runs end-to-end tests.
   Ex: ng e2e
9. extract-i18n - Extracts i18n messages.
   Ex: ng extract-i18n --output-path src/locale
10. generate (g) - Generates code from schematics.
    Ex: ng generate component login
11. lint - Runs linting tools.
    Ex: ng lint
12. new (n) - Creates a new Angular project.
    Ex: ng new my-app
13. run - Runs an Architect target.
    Ex: ng run my-app:build:production
14. serve (s, dev) - Builds & serves app with live reload.
    Ex: ng serve --open
15. test (t) - Runs unit tests.
    Ex: ng test
16. update - Updates dependencies.
    Ex: ng update @angular/core @angular/cli
17. version (v) - Displays Angular CLI version.
    Ex: ng version
